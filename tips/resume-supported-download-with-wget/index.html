Resume Supported Download with WGET

GNU WGET is a simple but powerful tool for downloading content from the web. Wget is non-interactive, meaning that it can work in the background, while the user is not logged on.

Posted by Karan on December 22, 2017   1088 views

 

GNU WGET is a powerful tool for downloading content from the web non-interactively.

wget https://www.python.org/ftp/python/3.6.4/Python-3.6.4.tar.xz

The above command will download Python-3.6.4.tar.xz in the current directory.

 

In case it interrupts use following commnad

wget --continue https://www.python.org/ftp/python/3.6.4/Python-3.6.4.tar.xz

 

To download it with different filename 

wget  -O  different_name.tar.xz   https://www.python.org/ftp/python/3.6.4/Python-3.6.4.tar.xz
